 # 🛠️ ابزارهای نگهداری ریپازیتوری

ابزاری برای پاک‌سازی و مدیریت حجم ریپازیتوری گیت‌هاب.

### اقدامات موجود

| اقدام | توضیحات |
|------|--------|
| **Clean Downloads Folder** | پاک کردن کامل پوشه دانلودها |
| **Clean Pages Folder** | پاک کردن آرشیو صفحات وب |
| **Clean Videos Folder** | پاک کردن ویدیوهای دانلود شده |
| **Sort Files in Downloads** | مرتب‌سازی فایل‌های تقسیم شده |

### نحوه استفاده

1. به **Actions** → **🛠 Maintenance Tools** بروید.
2. اقدام مورد نظر را انتخاب کنید.
3. روی **Run workflow** کلیک کنید.

**توصیه**: بعد از دانلود فایل‌های مورد نیاز، حتماً پوشه Downloads را پاک کنید.
