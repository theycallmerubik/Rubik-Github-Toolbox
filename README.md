# GitHub Free Tools Repository

**A powerful collection of GitHub Actions workflows for downloading, archiving, browsing, and remote access — completely free.**

---

## 🌟 About This Project

This repository provides a set of ready-to-use **GitHub Actions workflows** that turn public GitHub runners into powerful free tools:

- Download files from any direct URL
- Download YouTube videos (including restricted content)
- Archive entire web pages with screenshots and media
- Run a full remote desktop (RustDesk)
- Get temporary SSH access
- Maintain your repository automatically

**Built with ❤️ and hope for a free Iran** by [@TheyCallMeRubik](https://github.com/TheyCallMeRubik)

---

## ✨ Features

- **No server or VPS required** — everything runs on free GitHub Actions
- Automatic file splitting (for GitHub's 100MB file limit)
- Optional password protection (7z encryption)
- High success rate YouTube downloader with cookies support
- Web page archiving with screenshots and media
- Remote desktop and SSH access
- Easy maintenance tools

## 🚀 How to Use

### 1. Fork this Repository

- Click the **Fork** button at the top right
- Choose your account

### 2. Enable Workflows

After forking:
1. Go to **Actions** tab
2. Click on any workflow on the left
3. Click **"Enable workflow"** if prompted

### 3. (Optional but Recommended) Add YouTube Cookies

See the dedicated section in **[YouTube Downloader Guide](YouTube-Downloader-README.md#important-setting-up-youtube-cookies-recommended)**.

## 📌 Important GitHub Limitations

- **100 MB** maximum per file (workflows automatically split larger files)
- **5 GB** soft repository size limit
- **6 hours** maximum workflow runtime
- Public repositories are **free** but visible to everyone

**Regularly run the Maintenance workflow** to keep your repository healthy.
