# 🖥️ SSH Access via tmate

Get temporary SSH access to a GitHub Actions runner.

### How to Use

1. Go to **Actions** → **🖥️ SSH to Runner (tmate)**
2. Click **Run workflow**
3. (Optional) Set `limit_access` to `true` (recommended — only you can connect)
4. Click **Run workflow**

### Connection
The workflow will show a `ssh` command. Click on it or copy it to connect.

**Useful for**:
- Debugging other workflows
- Manual file operations
- Running custom commands
