# ⬇️ دانلود از URL

ابزار قدرتمند دانلود هر نوع فایلی از لینک مستقیم و ذخیره آن در ریپازیتوری گیت‌هاب.

### نحوه استفاده

1. به تب **Actions** در ریپازیتوری خود بروید.
2. workflow با نام **⬇️ Download from url** را انتخاب کنید.
3. روی **Run workflow** کلیک کنید.
4. فیلدها را پر کنید:

   - **urls**: لینک‌های دانلود با فاصله از هم جدا شوند  
     مثال: `https://example.com/file.zip https://example.com/video.mp4`
   - **mode**: 
     - `normal` → ذخیره به صورت معمولی (پیشنهادی)
     - `zip` → همیشه فایل را فشرده کند
   - **password** (اختیاری): رمز عبور برای فایل 7z

5. روی **Run workflow** کلیک کنید.
