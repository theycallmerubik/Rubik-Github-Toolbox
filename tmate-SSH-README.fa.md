# 🖥️ دسترسی SSH با tmate

دسترسی موقت SSH به_runner گیت‌هاب.

### نحوه استفاده

1. به **Actions** → **🖥️ SSH to Runner (tmate)** بروید.
2. **Run workflow** را بزنید.
3. گزینه `limit_access` را روی `true` بگذارید (توصیه می‌شود).
4. دستور `ssh` نمایش داده شده را کپی کنید و متصل شوید.

**کاربرد**: دیباگ کردن، اجرای دستورات دستی، رفع مشکل workflowها.
