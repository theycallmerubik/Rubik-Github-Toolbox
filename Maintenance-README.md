# 🛠️ Repository Maintenance Tools

Keep your repository clean and under GitHub size limits.

### Available Actions

| Action                     | Description |
|----------------------------|-------------|
| **Clean Downloads Folder** | Removes everything in `downloads/` |
| **Clean Pages Folder**     | Removes everything in `pages/` |
| **Clean Videos Folder**    | Removes everything in `videos/` |
| **Sort Files in Downloads**| Organizes split `.7z` parts into folders |

### How to Use

1. Go to **Actions** → **🛠 Maintenance Tools**
2. Click **Run workflow**
3. Choose the desired action
4. Click **Run workflow**

**Tip**: Run "Clean Downloads Folder" after you have downloaded the files you need.
