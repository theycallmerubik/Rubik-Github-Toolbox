# مخزن ابزارهای رایگان گیت‌هاب

**مجموعه‌ای قدرتمند از ورکفلوهای GitHub Actions برای دانلود، آرشیو، مرور و دسترسی ریموت — کاملاً رایگان.**

---

## 🌟 درباره این پروژه

این مخزن مجموعه‌ای از **ورکفلوهای آماده GitHub Actions** را ارائه می‌دهد که runners عمومی گیت‌هاب را به ابزارهای قدرتمند و رایگان تبدیل می‌کنند:

- دانلود فایل از هر لینک مستقیم
- دانلود ویدیوهای یوتیوب (شامل محتوای محدود)
- آرشیو صفحات وب کامل با اسکرین‌شات و رسانه
- اجرای دسکتاپ remoto کامل (RustDesk)
- دسترسی SSH موقت
- نگهداری خودکار مخزن

**ساخته شده با ❤️ و امید به ایران آزاد** توسط [@TheyCallMeRubik](https://github.com/TheyCallMeRubik)

---

## ✨ ویژگی‌ها

- **بدون نیاز به سرور یا VPS** — همه چیز روی GitHub Actions رایگان اجرا می‌شود
- تقسیم خودکار فایل‌ها (برای محدودیت ۱۰۰ مگابایت گیت‌هاب)
- حفاظت با رمز عبور (رمزنگاری 7z)
- دانلودر یوتیوب با موفقیت بالا و پشتیبانی از کوکی
- آرشیو صفحات وب با اسکرین‌شات و فایل‌های رسانه
- دسترسی دسکتاپ remoto و SSH
- ابزارهای نگهداری آسان

## 🚀 نحوه استفاده

### ۱. فورک کردن این مخزن

- دکمه **Fork** را در بالا سمت راست بزنید
- حساب خود را انتخاب کنید

### ۲. فعال کردن ورکفلوها

بعد از فورک:
1. به تب **Actions** بروید
2. روی هر ورکفلو در سمت چپ کلیک کنید
3. در صورت نیاز روی **"Enable workflow"** کلیک کنید

### ۳. (اختیاری اما بسیار توصیه شده) افزودن کوکی یوتیوب

برای بهترین نتیجه در دانلود یوتیوب، بخش مربوطه را ببینید:  
**[راهنمای دانلودر یوتیوب](YouTube-Downloader-README.md#important-setting-up-youtube-cookies-recommended)**

## ⚠️ Disclaimer کامل

**این مخزن فقط برای استفاده آموزشی و شخصی است.**

- استفاده از این ورکفلوها ممکن است شرایط خدمات یوتیوب، گیت‌هاب یا سایر پلتفرم‌ها را نقض کند.
- دانلود محتوای دارای کپی‌رایت بدون مجوز ممکن است در کشور شما غیرقانونی باشد.
- گیت‌هاب محدودیت سختی برای حجم مخزن دارد (**حدود ۵ گیگابایت**). استفاده سنگین ممکن است منجر به تعلیق مخزن شود.
- نویسنده و contributors هیچ مسئولیتی در قبال سوءاستفاده، تعلیق حساب یا عواقب قانونی ندارند.
- استفاده کاملاً با مسئولیت خودتان است.

**لطفاً همیشه قوانین کپی‌رایت و شرایط پلتفرم‌ها را رعایت کنید.**

## 📌 محدودیت‌های مهم گیت‌هاب

- **۱۰۰ مگابایت** حداکثر حجم هر فایل (فایل‌های بزرگ‌تر خودکار تقسیم می‌شوند)
- **۵ گیگابایت** حد نرم حجم مخزن
- **۶ ساعت** حداکثر زمان اجرای هر ورکفلو
- مخازن عمومی رایگان هستند اما برای همه قابل مشاهده‌اند

**به طور منظم از ورکفلو نگهداری استفاده کنید** تا مخزن شما سالم بماند.
