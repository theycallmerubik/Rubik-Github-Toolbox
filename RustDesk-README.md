# 🔵 RustDesk Remote Desktop

Run a full remote desktop (XFCE + Firefox) on GitHub Actions runner using RustDesk.

### How to Use

1. Go to **Actions** → **🔵 RustDesk Remote Desktop**
2. Click **Run workflow**
3. Enter:
   - **password**: Minimum 6 characters (this will be your RustDesk password)
   - **resolution** (optional): e.g. `1920x1080`, `1280x720`

4. Click **Run workflow**

### After Starting
- Wait 1–2 minutes
- The workflow will display the **RustDesk ID**
- Use the ID + your chosen password to connect from any RustDesk client

**Session Duration**: Up to 6 hours.

**Use cases**: Browsing, testing, light development, accessing blocked sites, etc.
