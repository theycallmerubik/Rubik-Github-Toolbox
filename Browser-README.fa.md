# 🌐 مرورگر وب و آرشیوکننده

مرور هر وبسایتی و ذخیره کامل آن (اسکرین‌شات + HTML + فایل‌های رسانه‌ای) داخل ریپازیتوری.

### نحوه استفاده

1. به **Actions** → **🌐 Browser** بروید.
2. **Run workflow** را بزنید.
3. اطلاعات را وارد کنید:
   - **url**: آدرس وبسایت
   - **mode**:
     - `full` → اسکرین‌شات + رسانه‌ها (پیشنهادی)
     - `text-only` → فقط کد HTML
   - **zip_password** (اختیاری): رمزگذاری آرشیو

### نتیجه
- یک پوشه جدید در `pages/` ایجاد می‌شود.
- شامل اسکرین‌شات، فایل HTML و رسانه‌ها است.
- همه چیز به صورت ZIP فشرده می‌شود.

**کاربرد**: آرشیو صفحات مهم، محتوای پشت paywall، تحقیقات و غیره.
